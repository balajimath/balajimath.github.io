Research Highlights

2025 · Evolving Systems (Springer)

A hybrid jumping swarm optimization based local search approach for minimum weighted dominating set problem

The Minimum Weighted Dominating Set problem is a challenging graph optimization problem with applications in network design and communications. This paper proposes a novel discrete swarm optimization-based local search algorithm to effectively explore the solution space and avoid local optima.

Graph Optimization Swarm Intelligence

2025 · IEEE Conference Proceedings

A Jumping swarm optimization-based hybrid approach for solving the minimum dominating set problem

The Minimum Dominating Set problem is an NP-hard graph problem with applications in networks and data analysis; this paper introduces JPSMD, a hybrid algorithm combining PSO with a three-phase local search to efficiently explore solutions.
